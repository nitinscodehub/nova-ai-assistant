âš™ï¸ OFFLINE/ONLINE MODE
-----------------------

Offline Mode:
  - Uses Vosk for speech recognition (local)
  - Uses pyttsx3 for TTS (local)
  - Rule-based responses (no internet needed)

Online Mode:
  - Uses Google Speech API
  - Uses OpenAI API for ChatGPT-like responses (if API key set)
  - Requires internet connection

Auto Mode (default):
  - Tries online first, falls back to offline
  - Best of both worlds

ğŸ› TROUBLESHOOTING
------------------

GUI doesn't start:
  - Check Tkinter: python3 -c "import tkinter"
  - Linux: sudo apt-get install python3-tk

Voice not working:
  - Check microphone permissions
  - Test mic: arecord -d 5 test.wav
  - Check audio system

Code execution fails:
  - Check Python version (3.10+)
  - Verify code syntax
  - Check logs/assistant.log

OpenAI errors:
  - Verify API key in settings
  - Check internet connection
  - Falls back to rule-based automatically
